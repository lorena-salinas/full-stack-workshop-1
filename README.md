## Full-Stack Workshop 1 – Git Merge Conflict Activity
### (Activity Sheet: https://docs.google.com/document/d/1n3dyZ5rtwohpM5UZq58SH44MTjzFZRXyE9pv59T0sq4/edit?tab=t.0)

This repository is for an Intro to Web Dev Workshop1 (3/12/26) activity on Git collaboration. The instructor and students intentionally create a merge conflict in `menu.txt` by changing the same line in different copies of the repo. Students then resolve the conflict using the standard Git workflow: `git status`, `git pull`, manually editing conflict markers, `git add`, `git commit`, and `git push`. Along the way, they also practice basic command-line navigation using `ls` to list folders and `cd` / `cd ..` to move into the project directory and back out again.
